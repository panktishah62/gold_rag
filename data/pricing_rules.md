# Gold Jewelry Pricing Rules

## Base Gold Value

Base Gold Value = weight (grams) × gold_rate_per_gram × purity

Where:
- Purity for 24K = 0.999
- Purity for 22K = 0.916
- Purity for 18K = 0.75
- Purity for 14K = 0.585

---

## Wastage Cost

Wastage Cost = Base Gold Value × wastage_percentage

Example:
If wastage is 8%, multiply base gold value by 0.08.

---

## Making Charges

Making Charges may be:

Option A:
Making Charges = Base Gold Value × making_charge_percentage

Option B:
Making Charges = weight × making_charge_per_gram

---

## Final Price Formula

Final Price =
Base Gold Value
+ Wastage Cost
+ Making Charges

Taxes (if applicable) are added separately.

---

## Example Calculation

Given:
- Weight = 18 grams
- Karat = 22K
- Gold rate = ₹6000 per gram
- Wastage = 8%
- Making charge = 10%

Step 1:
Base Gold Value = 18 × 6000 × 0.916

Step 2:
Wastage Cost = Base Gold Value × 0.08

Step 3:
Making Charges = Base Gold Value × 0.10

Step 4:
Final Price = Base Gold Value + Wastage + Making Charges
